## 🔍 LIFF คืออะไร?

**LIFF** = **LINE Front-end Framework**

คือระบบที่ LINE เตรียมไว้ให้คุณสร้าง **เว็บแอปที่เปิดใน LINE** ได้

### ตัวอย่างการใช้งาน:

#### 🚫 ไม่มี LIFF (ตอนนี้):
```
User: "แคมเปญ"
Bot:  "📱 พิมพ์ 'สมัคร TEST2026'"
→ ไม่มีฟอร์ม ไม่มีลิงก์
```

#### ✅ มี LIFF (หลัง Setup):
```
User: "แคมเปญ"
Bot:  "👉 สมัครเลย: https://liff.line.me/xxx"
User: *คลิกลิงก์*
→ เปิดฟอร์มสวยๆ ใน LINE
→ กรอกข้อมูล + อัปโหลดรูป
→ Submit เสร็จ!
```

## 🚀 วิธี Setup LIFF (15 นาที)

### ขั้นตอนที่ 1: สร้าง LIFF App (5 นาที)

1. เข้า: https://developers.line.biz/console/
2. เลือก Channel ของคุณ
3. Tab "LIFF" → คลิก "Add"
4. กรอก:
   ```
   Name: Application Form - Autobot
   Size: Full
   URL: https://autobot.boxdesign.in.th/liff/application-form.html
   Scope: ✅ profile, ✅ openid
   ```
5. คลิก "Add" → จะได้ **LIFF ID** (เช่น: 1234567890-AbCdEfGh)

### ขั้นตอนที่ 2: Update Database (2 นาที)

```sql
-- Connect Cloud SQL
gcloud sql connect autobot-db --user=root \
  --project=autobot-prod-251215-22549

USE autobot_prod;

-- Update LIFF ID
UPDATE campaigns 
SET liff_id = '1234567890-AbCdEfGh'  -- เปลี่ยนเป็น LIFF ID ของคุณ
WHERE code = 'TEST2026';

-- Check
SELECT code, name, liff_id FROM campaigns;
```

### ขั้นตอนที่ 3: Test (3 นาที)

พิมพ์ "แคมเปญ" ใน LINE → ควรเห็นลิงก์:
```
👉 สมัครเลย: https://liff.line.me/1234567890-AbCdEfGh?campaign=TEST2026
```
